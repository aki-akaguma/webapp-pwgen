# Password Generator WebApp Specification

## Requirements

- **Functionality:** Generate and display strings for use as passwords.
- **Character Count Selection:** Ability to select the length of the generated password.
  - Options: 8, 12, 15, 16 characters.
- **Character Type Selection:**
  - Lowercase letters only: a, b, c...
  - Lowercase letters + numbers: (1, 2, 3...) + (a, b, c...)
  - Lowercase letters + numbers + symbols: (1, 2, 3...) + (a, b, c...) + (!, %, &, ...)
- **Bulk Generation:** Generate 8 strings at a time.
- **Copy to Clipboard:** Allow the user to select one of the 8 generated strings and copy it to the clipboard.
